# 🍫 Chocolate Sales SQL Analysis

A comprehensive SQL analytics project built using PostgreSQL to analyze chocolate sales data and generate business insights through advanced SQL queries.

## 📌 Project Overview

This project demonstrates real-world Data Analyst SQL skills by solving business problems using:

- Joins
- Aggregate Functions
- Window Functions
- CTEs (Common Table Expressions)
- Subqueries
- Date Functions
- CASE Statements

The analysis is performed on a sales database containing nearly 50,000 orders across multiple products, customers, and stores.

## 💡 Key Insights

- Airport stores generated the highest overall profit.
- More than 31K unique customers placed orders.
- Every product received at least one order.
- Window functions enabled efficient ranking and trend analysis.
- CTEs improved query readability and maintainability.
